Kidz-Apps
PHP & MySQL Web Application
Kidz-Apps is a full-stack PHP web application built to showcase and manage educational mobile apps for children. The project includes product listings, a blog system, order placement, and a contact form, with a focus on clean structure, database integration, and basic web security practices.
Features
•	Product listing and detail pages with database-driven content
•	Blog system with post creation, listing, and detailed views
•	Order placement system storing customer and product data
•	Contact form with Google reCAPTCHA verification and email notifications
•	Embedded YouTube demo videos for products
•	Reusable layout components for maintainable code structure
•	Responsive UI built with Bootstrap and custom CSS

Tech Stack
•	Backend: PHP (PDO)
•	Database: MySQL
•	Frontend: HTML, CSS, Bootstrap
•	Client-side: JavaScript, jQuery
•	APIs & Services: YouTube Data API, Google reCAPTCHA

Security Practices
•	Used PDO prepared statements to prevent SQL injection
•	Applied output escaping (htmlspecialchars) to reduce XSS risks
•	Basic input validation for form submissions
